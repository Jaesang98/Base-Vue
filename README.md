# Base-Vue

이 프로젝트는 기본적인 Vue 프로젝트 설정입니다.

**수정날짜:**
2024-01-14

# Directory

# Create

    1) npm create vite@latest

    2) Project Name 입력
        - base-vue

    3) Package Name 입력
        - base-vue

    4) Framework 선택
        - vue

    5) Variant 선택
        - TypeScript

# Start & Build

    - npm run dev
    - npm run build

# Install npm

    Axios (서버 통신)
        - npm install axios

    Pinia (상태 관리)
        - npm install pinia

    Vue Router (라우터)
        - npm install vue-router

    Vue I18n (다국어)
        - npm install vue-i18n

    eslint, prettier
        - npm i -D eslint prettier eslint-plugin-vue eslint-config-prettier eslint-plugin-prettier @typescript-eslint/eslint-plugin @typescript-eslint/parser
